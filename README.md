## Project Overview
This repository contains the **complete, production-ready source code** and **all required deliverables** for the DIT3408 Group Project – a modern, real-time **Library Management System** developed for Central Metropolitan College.

The system successfully digitizes all library operations using a beautiful, responsive web application built with **HTML, CSS, Vanilla JavaScript** and **Google Firebase** (Authentication, Firestore, Hosting). It was completed **2 days ahead of schedule** and **under budget**, achieving 100% of planned scope.

## Key Features
- Dual secure portals: **Administrator** & **Member (Student/Faculty)**
- Real-time book catalog with instant availability updates (Firestore onSnapshot)
- Admin: Add/Edit/Delete books, manage students, view borrowing records, contact verified book retailers
- Member: Search, borrow, return books, view personal history
- Book Retailers module with direct Email/Call buttons
- Fully responsive cosmic dark UI with glass-morphism design
- Live deployment on Firebase Hosting
